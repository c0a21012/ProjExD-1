# 第3回
## 迷えるこうかとん(ex03/maze.py)
### 追加機能
- スタートとゴール：スタート地点とゴール地点に色を塗り視覚化する
- ゴール時のこうかとん：ゴール地点に到達すると、こうかとんの画像が「8.png」から「6.png」に変化する
- 通った道の視認：一度通った道にこうかとんの画像を残し、どこを通ったか分かるようにする ゴールすると画像はクリアされる
